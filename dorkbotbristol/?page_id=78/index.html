I can’t see any video??

We will be streaming straight into this web page using the free Xiph Ogg/Theora video platform and the iTheora player courtesy of XtreamLab. The best way for you to see the video is with the excellent Cortado, for which you need to have Java installed. To test viewing with Cortado do the following:

  • In the video window to the left click on the top right hand icon labeled Options.
  • From the menu select Play with the Java plugin
  • If you have Java, then a message will appear asking you whether you wish to trust the certificate used to sign the applet, click Trust (trust us ;)).

We have tested the stream using most browser combinations, but if you do have trouble watching it in your browser, then try using either VideoLAN Client or QuickTime Player. You will then need to copy and open the address http://live.xtreamlab.net:8000/dorkbotbristol.ogg.
